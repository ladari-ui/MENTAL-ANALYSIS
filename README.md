# MENTAL-ANALYSIS
Mental Health Emotion Analyzer is a Machine Learning-based web application that detects emotions from both text and voice input using TF-IDF and Logistic Regression. The system provides real-time emotion predictions along with confidence scores through an interactive Streamlit interface.
